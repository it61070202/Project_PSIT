# Cars Sale Analysis (CSA)
วิเคราะห์ยอดขายรถยนต์แต่ละยี่ห้อในประเทศNorwayตั้งแต่ปี 2007-2016 ว่ารถยนต์ยี่ห้อไหนมีผู้ใช้งานมากสุด
วิเคราะห์ว่า Top 10 ของยอดขายรถยนต์ว่ามีรุ่นไหนบ้าง และรุ่นไหนขายดีที่สุด

# ข้อมูล
